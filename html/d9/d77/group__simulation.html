详细描述

根据物理模拟结果生成动画的模块。

Dandelion 允许设置每个物体的质量、速度、合外力三种物理属性，这些属性直接附加到相应的 Object 实例上，并在调用 Object::update 方法时更新该物体的位置和速度。点击动画模式工具栏上的 Start 按钮后，场景将进入播放状态，并在每一帧调用各物体的 update 方法，渲染位置更新后的物体。
