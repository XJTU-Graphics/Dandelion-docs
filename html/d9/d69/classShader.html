详细描述

对 GLSL Shader 的简单封装。

一个 Shade 对象在构造时并不直接加载 GLSL 代码，正确的使用顺序是：

  1. 调用 load_vertex_shader
  2. 调用 load_fragment_shader
  3. 调用 compile 编译 Shader
  4. 前三个步骤都返回 true 的前提下，调用 use 使用 Shader
