软光栅渲染器

这个类主要由光栅化所需的各种属性（如成像平面的长宽，以及M,V,P矩阵等），以及光栅化 整个流程所需的各种操作对应的函数构成（如vertex shader, fragment shader,等）

对整个物体进行光栅化渲染

对物体的所有三角形面片进行遍历，对三角形的顶点应用vertex shader，对顶点的坐标和法线方向 进行变化；同时获取三角形每个顶点在view space下的坐标便于后续的插值操作；对顶点的法线同样 需要变换到view space下用于后续的插值，最后执行光栅化

参数
TriangleList存储了当前渲染物体的所有三角形面片
material当前渲染物体的材质
lights存储了当前场景中的所有光源
camera当前使用的相机（观察点）

Camera  camera 
)
private

光栅化当前三角形

整个光栅化渲染应当包含：VERTEX SHADER -> MVP -> Clipping -> VIEWPORT -> DRAWLINE/DRAWTRI -> FRAGMENT SHADER 在进入rasterize_triangle之前已经完成了VERTEX SHADER到VIEWPORT的整个过程，可以被看作几何阶段 在进入rasterize_triangle之后，就开始进行真正的光栅化了，逐片元的进行着色

参数
t当前进行光栅化的三角形
world_pos三角形三个顶点的坐标(已经变换到world_space下)
material当前三角形所在物体的材质
lights当前场景的所有光源(已经变换到world_space下)
camera当前使用的相机（观察点）
