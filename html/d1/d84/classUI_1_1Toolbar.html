◆ select_object_by_id()

void Toolbar::select_object_by_id ( Scene &  scene,
std::optional< std::size_t >  object_id 
)

根据指定的物体 ID 设置场景中的被选物体。

参数
object_id为空 (std::nullopt) 表示清空选择结果， 清空后场景处于没有任何物体被选中的状态；非空表示选择相应 ID 的物体。 若场景中不存在该 ID 对应的物体，结果等同于 ID 为空。
