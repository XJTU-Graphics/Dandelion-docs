详细描述

在这个头文件中，定义了有关 GUI 的一些通用配置， 它被所有与 GUI 相关的 源文件 (.cpp) 共享。

换言之，这个文件与各种 UI 组件的定义无关，所以不应该被任何头文件使用 #include 包含。

函数说明

◆ px()

float px ( float  logical_pixel)
inline

将逻辑像素（与设备无关的尺寸）转换为物理像素，例如用 px(200.0f) 表示逻辑上的 200 像素。

返回
物理像素乘以缩放系数。
