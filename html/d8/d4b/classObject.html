详细描述

表示物体的类。

一个逻辑上的“物体”是若干个 Mesh 组成的，所有的 Mesh 共享 Model 矩阵，在布局模式下不可分离。

成员函数说明

◆ model()

Matrix4f Object::model ( )

此物体的 Model 矩阵，对应模型变换。

类成员变量说明

◆ meshes

std::vector<GL::Mesh> Object::meshes

每个 Mesh 存储的都是该物体的模型坐标系下的顶点坐标。 如需获取世界坐标系下的坐标，请乘上 Model 矩阵。
