详细描述

表示物体的类。

“物体”由存储几何数据的 mesh、存储变换信息的位姿参数和存储颜色的材质组成， 每个 Object 实例对应唯一的 GL::Mesh、GL::Material 和唯一的模型变换矩阵 (Model Transform Matrix)，是场景中的最小单元。

由于 Object 对象持有的 GL::Mesh 实例是不可复制的，当前的 Object 也是不可复制的。但“复制一个物体”在逻辑上完全合理，因此未来会实现该类的复制构造函数。
